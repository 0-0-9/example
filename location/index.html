location 的属性
属性名 示例 说明 查看本网站
hash "#contents" 设置或返回从井号 (#) 开始的 URL（锚）
host "www.wrox.com:80" 设置或返回主机名和当前 URL 的端口号
hostname "www.wrox.com" 设置或返回不带端口号的服务器
href "http://www.wrox.com" 设置或返回当前加载页面的完整URL。结果同location.toString()
pathname "/WileyCDA" 设置或返回URL中目录和（或）文件名
port "8080" 设置或返回端口号。若无端口号，返回空字符串
protocol "http:" 设置或返回页面使用的协议
search "?q=js" 设置或返回URL的查询字符串。以问号开头
